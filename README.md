### Installation: 

1. Launch Minecraft
2. Go to Options → Resource Packs
3. Click Open Pack Folder
4. Drag your .zip file into the folder
5. Enable the pack
6. Enter a world and enchant a diamond sword with Fire Aspect or any of the supported enchants

If everything is correct, the texture should change.
